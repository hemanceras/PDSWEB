ETIQUETA DESCRIPCIÓN
body Delimita el cuerpo del documento, debe ir recogido dentro de las etiquetas de apertura y cierre de body.
head Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. Delimita el encabezado del documento.
title Usada para definir el título de la página web. Se muestra en la pestaña del navegador.
style Usadas para introducir código CSS en línea, es decir, en el propio documento HTML.
h1,h2,h3,h4,h5,h6 Son usadas para jerarquizar el contenido de la web. Las etiquetas se usan para explicar brevemente el contenido que irá a continuación.
img para "pintar" una imagen en la página web.
table Para la apertura y cierre de una tabla.
tr Se usa para indicar una fila de celdas de una tabla.
audio Usada para cargar en una web un archivo de audio o stream de audio.
video Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.
