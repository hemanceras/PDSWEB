Numero ETIQUETA DESCRIPCIÓN
1. body Delimita el cuerpo del documento, debe ir recogido dentro de las etiquetas de apertura y cierre de body.
2. head Representa una colección de metadatos acerca del documento, incluyendo enlaces a, o definiciones de, scripts y hojas de estilo. Delimita el encabezado del documento.
3. title Usada para definir el título de la página web. Se muestra en la pestaña del navegador.
4. style Usadas para introducir código CSS en línea, es decir, en el propio documento HTML.
5. h1,h2,h3,h4,h5,h6 Son usadas para jerarquizar el contenido de la web. Las etiquetas se usan para explicar brevemente el contenido que irá a continuación.
6. img para "pintar" una imagen en la página web.
7. table Para la apertura y cierre de una tabla.
8. tr Se usa para indicar una fila de celdas de una tabla.
9. audio Usada para cargar en una web un archivo de audio o stream de audio.
10. video Se usa para reproducir video en la página web junto a sus archivos de audio y capciones asociadas.

Numero ESTILOS DESCRIPCIÓN
1. Color Define el codigo de color de la letra en los estilos estilo. Ejemplo: color: #040404;
2. Background Controla el color de fondo. Ejemplo: background-color: #F0B429;
3. Border No son visibles, y es como un cuadro imaginario que envuelve todo el contenido del bloque. Ejemplo: border-width:2px; border-color:blue; border-style:solid;
4. Margin Son el espacio comprendido entre el bloque y el borde de la ventana activa del navegador. Ejemplo: margin-top:2px; margin-left:2px;
5. Text Se definen las propiedades del texto en la pagina. Ejemplo: text-decoration:underline; text-transform:capitalize; text-align:left; text-indent:30px;
6. Font Se establecen caracteristicas de algunos estilos, como tamaño, tipo de lectra, fuente, entre otras. Ejemplo: font-size:20px; font-family:Courier; font-weight:bold; font-style:italic;
7. Height Tamaño de altura. Ejemplo: height:100px;
8. Width Tamaño de ancho. Ejemplo: width:100px;
9. link Establecer enlaces. Ejemplo: LINK REL="stylesheet" TYPE="text/css" HREF="estilo.css"
10. list Definir tipos de listas. Ejemplo: ul {list-style-type:square;} ol {list-style-type:upper-roman;}
11. table Definir tipos de listas. Ejemplo: ul {list-style-type:square;} ol {list-style-type:upper-roman;}
11. table Definir caracteristicas de las tablas. Ejemplo: table-row-group, table-header-group, table-footer-group, table-row, table-column-group, table-column, table-cell, table-caption
12. position Indica la posición de un bloque. Ejemplo: div style="position: relative; top: 5px; left: 5px;
13. Overflow Regula la visivilidad de los contenidos. Ejemplo: overflow:hidden; scroll; visible;
14. Float Para la alineación de los bloques. Ejemplo: img {float: right;}
